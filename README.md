AI Based Smart Eye for Blind People

Project Description

AI Based Smart Eye for Blind People is an assistive technology system designed to help visually impaired individuals understand their surroundings.
The system uses Artificial Intelligence, Computer Vision, and Raspberry Pi to detect objects, recognize faces, identify currency, and provide audio feedback to the user.

A mobile Android application is also used to manage and communicate with the device through Bluetooth.

Features

- Object detection using AI and Computer Vision
- Face recognition
- Currency detection
- Text-to-speech voice alerts
- Bluetooth communication with mobile app
- Real-time environment assistance for visually impaired users

System Architecture

The system consists of two main components:

Raspberry Pi Module

Responsible for AI processing and vision detection.

Main files:

- "main.txt" – Main program controlling the system
- "object_detection.txt" – Detects surrounding objects
- "face_recognition.txt" – Recognizes stored faces
- "currency_detection.txt" – Identifies currency notes
- "text_to_speech.txt" – Converts system output to voice alerts
- "bluetooth_server.txt" – Handles Bluetooth communication
- "face_data_manager.txt" – Stores and manages face data
- "shutdown.txt" – System shutdown control

Android Application

Provides user interface and communication with the device.

Main files:

- "MainActivity.txt"
- "BluetoothHelper.txt"
- "CaptureImagesActivity.txt"
- "FacesActivity.txt"
- "SyncActivity.txt"
- "MainUI.txt"
- "CaptureUI.txt"
